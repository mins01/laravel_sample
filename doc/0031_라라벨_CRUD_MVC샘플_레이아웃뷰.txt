*. 레이아웃 뷰를 사용하는 이유
일관적인 모양을 유지하기 위해서.
필수 스크립트 로드 등.
사용 안할 경우 각 페이지에서 이 부분을 처리해야한다.

*. 레이아웃 뷰 위치
sample\resources\views\layouts

*. 레이아웃 설명
default.blade.php  :  기본형, navbar 등을 포함
empty.blade.php  :  빈 페이지용 (필수 JS와 CSS 포함)
blank_html.blade.php  :  순수한 빈 HTML 페이지용 (필수 JS와 CSS 미포함. 순수한 HTML)
blank_text.blade.php  :  순수한 빈 TEXT 페이지용
